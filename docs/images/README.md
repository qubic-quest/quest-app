## Screenshot Tips

- Use high-quality screenshots (at least 1280px wide)
- Show real data examples when possible
- Capture the full UI component including headers
- Use light mode for consistency
- Highlight key features in each screenshot
- Include diverse query examples

## Taking Screenshots

1. Start the development server: `npm run dev`
2. Navigate to http://localhost:3000
3. Ask various questions to trigger different tools
4. Take screenshots of the results
5. Save them in this directory with the exact filenames listed above
6. Optimize images for web (consider using ImageOptim or similar)
